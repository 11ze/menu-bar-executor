# 应用图标设计

## 背景

menu-bar-executor 是一个 macOS 菜单栏命令执行器应用，目前缺少一个专业的应用图标。需要一个符合 macOS 设计规范、能体现应用功能的图标。

## 设计方向

基于用户选择：
- **设计风格**：线性插画（细线条勾勒，精致优雅）
- **视觉元素**：菜单栏条 + 光标
- **配色方案**：深色主题（深灰/黑色底，适合暗色模式）

## 图标方案

### 方案 A：简约菜单栏
- 顶部细长菜单栏条
- 光标指针指向菜单项
- 纯深色背景 + 白色/浅灰线条
- 圆角矩形 App Icon 轮廓

### 方案 B：终端融合
- 菜单栏条 + 终端窗口元素结合
- 光标 + 命令提示符 `$`
- 渐变深色背景

### 方案 C：动态执行
- 菜单栏条 + 闪电/执行符号
- 光标正在点击
- 强调"执行"动作

## 执行步骤

1. 使用 canvas-design skill 生成 3 个图标选项（PNG 格式）
2. 用户选择最喜欢的方案
3. 如需调整，根据反馈修改
4. 将最终图标集成到项目中（配置 Info.plist 和 Assets）

## 输出文件

- `AppIcon-Option-A.png`
- `AppIcon-Option-B.png`
- `AppIcon-Option-C.png`
- 尺寸：1024x1024（macOS App Icon 标准）

## 验证

生成后用户确认选择，再集成到项目中。
