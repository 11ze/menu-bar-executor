# 统一通知标题

## Context

用户希望所有通知的标题统一为 "Menu Bar Executor"，而不是当前的 "✅ 执行成功" 或 "❌ 执行失败"。

当前实现：
- `showSuccess`: title="✅ 执行成功", subtitle=命令名
- `showFailure`: title="❌ 执行失败", subtitle=命令名
- `showReloadSuccess`: title="Menu Bar Executor"（已符合要求）

## 修改方案

修改 `Sources/App/NotificationManager.swift`，调整通知内容结构：

### showSuccess
```swift
content.title = "Menu Bar Executor"
content.subtitle = "✅ \(commandName)"
content.body = "输出：\n\(output)"  // 保持不变
```

### showFailure
```swift
content.title = "Menu Bar Executor"
content.subtitle = "❌ \(commandName)"
content.body = "错误：\n\(error)..."  // 保持不变
```

## 修改文件

- [NotificationManager.swift](Sources/App/NotificationManager.swift) - 第 31-32 行和第 51-52 行

## 验证方式

1. 构建并运行应用
2. 执行一个成功的命令，检查通知标题是否为 "Menu Bar Executor"
3. 执行一个会失败的命令，检查通知标题是否为 "Menu Bar Executor"
